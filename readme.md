# Bootstrap 4 for WordPress

Minimalistic Wordpress theme on Bootstrap 4 (alpha). This theme is based on Bootstrap Four (https://wordpress.org/themes/bootstrap-four/). Modernized using OOP.

## Features (So Far)

* Pagination
* Menu Walker
* Core Bootstrap4
* Custom Comments form
* Font Awesome
* Favicon Support
